# Hypermarket Holiday Sales Forecasting

Forecasting Walmart sales during Thanksgiving and Christmas using Facebook Prophet.

## Project Overview

This repository contains the code used to analyze and forecast hypermarket sales
during the two largest U.S. retail holidays. This version contains only the 
GitHub infrastructure (no dataset, no figures, no notebooks) as requested.

## Structure

- .github/workflows/ci.yml  — CI workflow
- docs/                     — GitHub Pages documentation
- src/                      — Source code package
- Makefile                  — Common commands
- setup.py                  — Package installer
- requirements.txt          — Dependencies
- LICENSE                   — MIT license
- .gitignore                — Ignore rules
